# EZUIKit-player-controls

![Build Demos](https://github.com/Ezviz-OpenBiz/EZUIKit-player-controls/workflows/build/badge.svg)

该项目用来存储 ezuikit 播放器相关的 UI 控件。

## Use

控件都存储在 `packages`下， 仅需要改 `packages` 下的文件， 其他文件不要动


## npm publish

npm 包的发布仅需要打一个 git tag 然后提交， github actions 的 release.yml 会自动发布

注意： 新的npm包需要在 npm 包的设置中配置 github actions
